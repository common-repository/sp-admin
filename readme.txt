A clean and simple Wordpress Administration template, putting a fresh and more modern spin on the current administration template.

== Description ==

This plugin enables a new theme for your wordpress administration area. It is a neat and subtle twist on the current wordpress administration theme. For a more modern administration area of your website.

== Installation ==

1. Upload the `sp-admin` folder to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress - Look for SP Admin and click activate.
